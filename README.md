TalentScout is an interactive and user-friendly Streamlit-based chatbot designed to assist with the hiring process. It collects candidate information, anonymizes sensitive details, and generates technical questions tailored to the candidate's expertise and experience. The chatbot is visually appealing, modern, and easy to use, making it a great tool for recruiters and HR professionals. ğŸ˜ŠğŸ˜ŠğŸ˜Š
